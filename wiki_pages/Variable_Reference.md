* **[Variable Reference: Core](Variable_Reference_Core)**: Variables for core system roles including bootstrap, hardware support, and system services

* **[Variable Reference: Networking](Variable_Reference_Networking)**: Variables for networking roles including firewalls, VPN, and container networks
* **[Variable Reference: Storage](Variable_Reference_Storage)**: Variables for storage roles including backup and filesystem management

This categorization helps organize the numerous variables used throughout the Deploy-System-Unified project, making them easier to find and reference.
