# Example: Using Qwen Sub-Agents

This directory contains examples demonstrating how to use the Qwen sub-agent system.

## Available Examples

### 1. Basic Agent Usage
- `basic_example.py` - Demonstrates initializing the agent manager and querying agents

### 2. Task Routing
- `task_routing.py` - Shows how to route tasks to appropriate agents based on capabilities

### 3. Research Task
- `research_example.py` - Example of using the research agent for information gathering

### 4. Code Generation
- `coding_example.py` - Example of using the coding agent for code generation tasks

## Running Examples

To run the examples, make sure you have the required dependencies installed:

```bash
pip install openai
```

Then run any example:

```bash
python basic_example.py
```

Make sure your Qwen API key is set in the environment variable `QWEN_API_KEY`.
